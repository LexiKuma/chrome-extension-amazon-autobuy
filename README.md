## Amazon自动刷单插件
* AutoChangeIP_20150513.Q 这个是给按键精灵用的脚本，主要用来操作代理软件，然后代理软件切换ip，为了防止刷单检测
* 老版本视频：[查看视频](https://zhoukekestar.github.io/chrome-extension-amazon-autobuy/index.html)
